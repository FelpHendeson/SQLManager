# Fluxo principal

1. **Colagem de SQL**: o usuário cola ou digita a consulta no editor principal, iniciando a sessão de análise.
2. **Análise**: o sistema valida a sintaxe, identifica pontos de melhoria (performance, legibilidade, padronização) e prepara sugestões.
3. **Sugestões/saída + aplicação de correções**: o usuário revisa as sugestões e aplica as correções usando o botão **“Aplicar formatação”**, gerando a saída final formatada.
